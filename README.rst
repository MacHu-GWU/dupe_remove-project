Install
------------------------------------------------------------------------------

``dupe_remove`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install dupe_remove

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade dupe_remove
